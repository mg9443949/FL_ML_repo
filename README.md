# 🔗 FL_ML_repo

## 📌 About the Project
This repository contains the Machine Learning modules developed as part of **FlowLock**, a joint project focused on building a productivity-enhancing system that detects and minimizes user distractions.

> ⚡ Note for Recruiters:  
> This is part of a **collaborative project (FlowLock)**. This repository specifically showcases the ML components I contributed/worked on within the larger system.

---

## 🚀 What is FlowLock?
**FlowLock** is a smart productivity system designed to help users stay focused by detecting distractions in real time using AI + hardware integration.

It combines:
- Computer Vision 👁️  
- Audio Processing 🎤  
- Gesture Recognition ✋  
- Authentication Systems 🔐  

---

## 🧠 ML Features Implemented

### 👤 Face-Based Attention Detection
- Detects presence and monitors attention  
- Identifies if user is distracted or not  

### 🔊 Sound-Based Distraction Detection
- Analyzes ambient noise levels  
- Flags distracting environments  

### ✋ Hand Gesture Control
- Detects gestures to control system behavior  
- Enables touch-free interaction  

### 🔐 Face Authentication
- Registers and verifies authorized users
